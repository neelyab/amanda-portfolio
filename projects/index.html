Thread Color Palette Generator

The thread palette generator app is a full-stack application that helps the user create beautiful thread color palettes. There are plenty of color palette tools out there, but not ones that color-match embroidery thread, so this app does it for you. Upon creating an account, the user can select a base color, the number of colors, and the color scheme to generate a palette. They are also able to save the palettes to their account.

Technologies used: HTML, CSS, Javascript, React, Node.js, Express, PostgreSQL
