Embroidery Helper App

The hand embroidery app is a full-stack application that helps the user learn how to embroider. It was built to help a user with little to no embroidery experience to learn embroidery stitches and apply their new skills by creating embroidery projects. Upon creating an account, the user can search for embroidery stitches and projects to learn, as well as save and delete embroidery stitches and projects to their account.

Technologies used: HTML, CSS, Javascript, React, Node.js, Express, PostgreSQL

Meal Planner App

The meal planner app is a full-stack application that helps people that have different diets to find recipes and plan their meals together. It can be hard to share a meal with someone that has a different diet, but the meal planner app makes it possible. Upon creating an account, the user can search for recipes and save these recipes in customized meal plans. The app uses Edamam's recipe API for the recipes and an API built from scratch for user info and saved recipes.

Technologies used: HTML, CSS, Javascript, React, Node.js, Express, PostgreSQL

Trail Running Planner App

The trail running planner app helps runners find trails and plan their runs easier. The app was built to allow the user to conveniently find trails and also see the weather forecast all in one place. The user submits their address and the maximum distance they would like to travel to the trail. The app then displays current weather, 7-day forecast, and trails within the selected distance.

Study App

The study app is based on the Pomodoro technique. Traditionally, it uses a tomato timer, but this app uses an egg timer. The app was built to encourage and help students to keep track of how long they study. The user selects the amount of time they would like to study and a time countdown displays on the screen. Fried eggs are accumulated after each study session is complete.

Technologies used: HTML, CSS, Javascript, React
